# JAP\_LOCAL\_PYTHON #

## PRE-INSTALLATION ##
  * install PYTHON >= 2.7.0
  * install TWISTED
  * install PYOPENSSL

## INSTALLATION ##
  * unzip JAP\_LOCAL\_PYTHON-X.X.X.zip
  * open JAP\_LOCAL\_PYTHON-X.X.X
  * [configure JAP\_LOCAL.json](CONFIGURE_JAP_LOCAL.md)
  * execute JAP\_LOCAL.bat
  * [configure BROWSER](CONFIGURE_BROWSER.md)

## IMPORTANT ##
