# JAP\_REMOTE\_WS\_NODE #

## PRE-INSTALLATION ##
  * install NODE >= 0.8.0

## INSTALLATION ##
  * unzip JAP\_REMOTE\_WS\_NODE-X.X.X.zip
  * open JAP\_REMOTE\_WS\_NODE-X.X.X
  * execute NM.bat
  * [configure JAP\_REMOTE\_WS.json](CONFIGURE_JAP_REMOTE_WS.md)
  * execute JAP\_REMOTE\_WS.bat

## IMPORTANT ##
